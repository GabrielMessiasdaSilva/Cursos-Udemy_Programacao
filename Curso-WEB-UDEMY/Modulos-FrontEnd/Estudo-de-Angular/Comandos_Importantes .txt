///////// INSTALA O ANGULAR CLI GLOBALMENTE://///////
npm install -g @angular/cli 
npx @angular/cli new Angular 


/////////INICIA O PROJETO////
ng new Nome-PROJETO

/////////RODAR O PROJETO//////////
cd [PROJECT NAME]
ng serve    
///////////////////////////////
ng g c card (cria novos componente)

Caso você esteja utilizando a versão 17 do Angular, eu peço que você utilize o seguinte script para gerar um novo projeto:



ng new NOME_DO_PROJETO --no-standalone
